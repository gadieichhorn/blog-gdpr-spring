GDPR Chat

Demo chat (Vertx.io) application demonstrate how to handle persistent queue messages with sensitive user data (like support chat)

Chat records that persist into a queue (kafka) are very hard to clean is a user wish to be forgotten.

This demo app persist the chat messages to a queue with encryption, this way we can delete the user private key making all user chat messages inaccessible
